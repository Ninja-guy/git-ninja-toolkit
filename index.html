Git Ninja Toolkit (GNT)

Do you want to be a Total Git Badass? Do you want your colleagues to revere you as a Git god? Then you'll want to use the Git Ninja Toolkit (GNT)!

GNT contains 1337 Git configuration files that will probably boost your Git productivity by a non-trivial amount! (Note, I make no guarantees that the toolkit will actually boost your productivity, by any amount).

GNT also contains configuration files for some text-editors that you might use in your daily software development.
